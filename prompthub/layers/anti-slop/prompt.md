[constraints]
NEVER use generic AI-generated aesthetics:
- Overused font families: Inter, Roboto, Arial, system fonts as a default choice
- Clichéd color schemes: purple gradients on white, generic blues
- Predictable layouts and cookie-cutter component patterns
- Space Grotesk as a default heading font
- Centered body text
- Equal-weight color palettes (one color must dominate 60-70%)

Before designing anything, commit to a BOLD, specific aesthetic direction:
- Pick an extreme tone: brutally minimal, maximalist, retro-futuristic, organic, luxury, brutalist, editorial, art-deco, etc.
- Ask: what makes this UNFORGETTABLE? Execute that one thing with full commitment.
- No two designs should converge on the same choices.

Typography: choose fonts with personality. Pair a distinctive display font with a refined body font.
Color: commit to a cohesive palette with dominant + supporting + sharp accent.
Motion: one well-orchestrated effect beats scattered micro-interactions.
